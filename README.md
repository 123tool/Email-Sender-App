# Email Sender

A simple email sender application built with Python and Tkinter.

## Features
*   Send emails with subject and message
*   Login and logout functionality
*   Error handling for invalid email and password
*   Validates email address and password

## Installation
### Termux (Android)
1.  Install Python and pip: `pkg install python`
2.  Install required packages: `pip install pillow`
3.  Clone the repository: `git clone https:                                            
4.  Navigate to the repository: `cd email-sender`
5.  Run the application: `python email_sender.py`

### Windows
1.  Install Python and pip from the official website: <https://www.python.org/downloads/>
2.  Install required packages: `pip install pillow`
3.  Clone the repository: `git clone https:                                            
4.  Navigate to the repository: `cd email-sender`
5.  Run the application: `python email_sender.py`

## Usage
1.  Enter your email address and password
2.  Click the "Login" button
3.  Enter the recipient's email address, subject, and message
4.  Click the "Send" button

## Notes
*   Make sure to enable "Less secure apps" in your Google account settings to use this application.
*   This application uses the SMTP protocol to send emails.
*   Please use this application responsibly and do not spam others.
